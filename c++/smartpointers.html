1.1 Operations to shared_ptr

shared_ptr<T> sp
Null smart pointer that can point to objects of type T.
*sp
Dereference sp to get the object to which sp points.
sp->mem
Synonym for (*sp).mem.
sp.get()
Return the pointer in p. The object to which the returned pointer points will disappear when the smart pointer deletes it.
sp.swap(q) and swap(sp, q)
Swaps the pointers in sp and q.
make_shared<T>(args)
Returns a shared_ptr pointing to a dynamically allocated object of type T. Use args to initialize that object.
shared_ptr<T> p(q)
p is a copy of the shared_ptr q, increaments the count in q. The pointer in q must be convertible to T*.
p = q
p and q are shared_ptrs holding pointers that can be converted to one another. Decrements p's reference count and increaments q's count, deletes p's existing memory if p's count goes to 0.
p.unique()
Returns true if p.use_count() is one, false otherwise.
p.use_count()
Returns the number of objects sharing with p.
